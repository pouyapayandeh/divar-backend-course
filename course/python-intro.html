# آشنایی با پایتون

پایتون یکی از زبان‌های محبوب جهانه که توی شرکت خیلی ازش استفاده می‌کنیم. پایتون زبانیه که خیلی سریع می‌شه باهاش کد نوشت

# برای مطالعه

  • برای آشنایی با پایتون پیشنهاد می‌کنیم این کتاب رو از اول تا آخر فصل ۶ (به جز فصل ۵) بخونی.
  • استاندارد pep8 (قواعد ظاهری زبان پایتون) رو خوبه که مطالعه کنی. (البته IDEهایی مثل PyCharm خودشون این استانداردها رو می‌تونن اعمال کنن و خوبه که از اونا استفاده کنی)

# تمرین

  1. نرمالایزر عدد: تابعی بنویس که یک رشته بگیره و تمام رقم‌هایی که در اون رشته وجود داره رو انگلیسی کنه. یعنی ۲ تبدیل بشه به 2 و 3، 3 باقی بمونه. ۴ و ٤ هم بشن 4.
  2. یه برنامه بنویس که از stdin بتونه کوئری‌های زیر رو بخونه و هندل کنه:
    • تسکی به نام x بساز!
    • یوزری به نام y بساز!
    • تسک x رو به یوزر y اختصاص بده!
    • لیست تمام تسک‌هایی رو که یوزر y داره، چاپ کن!
    • لیست تمام یوزرهایی که تسک x بهشون اختصاص داده شده رو چاپ کن!

مثال:

CREATE USER X

CREATE TASK Y

ASSIGN X Y

LIST USER Y

LIST TASK X

  1. کد تمرین قبل رو طوری تغییر بده که اجرای اون با help-- نحوه‌ی ورودی گرفتن رو چاپ کنه و با n- تعداد دستورات رو بگیره.

  2. جداکننده ۳ رقمی: تابعی بنویس که یک عدد بگیره و یک رشته برگردونه که ۳ رقم، ۳ رقم جداشده‌ی اون عدده.

  3. یه برنامه بنویس که لیست تمام فایل‌های درون یک فولدر رو توی فایلی به نام index (توی همون پوشه) بنویسه. این فولدر ممکنه تا چند سطح داخل خودش فولدرهای دیگه داشته باشه و اون فایل‌ها هم باید توی index لیست بشن.

  4. برنامه‌ای بنویس که از ورودی آدرس یک فایل ورودی و یک فایل خروجی رو بخونه. هر خط فایل ورودی یک شیء jsonئه. توی فایل خروجی باید همون ورودی‌ها رو بر حسب ِ id مرتب شده بنویسی. (json در پایتون)

    نمونه محتویات داخل فایل ورودی:

{"id": 1, "name": "gholam"}
{"id": 11, "name": "somayye"}
{"id": 4, "age": 22, "city": "tehran"}
{"id": 666, "name": "satan", "age": 1000000}

# پکیج در پایتون

ماژول‌ها و توابعی در کد زدن با پایتون بعضا لازممون می‌شه که همه‌شون از ابتدا وقتی پایتون رو نصب می‌کنیم قابل استفاده نیستن. بعضیاشون تحت چیز‌هایی به اسم پکیج توسط آدم‌های مختلف نوشته‌شدن و قابل استفاده‌ن. با استفاده از برنامه‌هایی مثل پیپ (pip) می‌تونی این پکیج‌ها رو نصب کنی. برای شروع خوبه که pip رو از اینجا نصب کنی.

هر پروژه‌ای ممکنه به یه سری پکیج‌های پایتون نیاز داشته باشه، ممکنه دو تا پروژه به دو تا ورژن مختلف از یک پکیج نیاز داشته باشن و این دو تا پروژه با نسخه‌های دیگهٔ اون پکیج کار نکنن، واسه همین خوبه که یه جوری بتونیم پکیج‌هایی که برای هر پروژه نصب می‌کنیم رو جدا کنیم. برای این کار از virtualenv استفاده می‌کنیم.

برای استفاده‌ی راحت‌تر از virtualenv می‌تونی از virtualenvwrapper استفاده کنی.

یک virtualenv برای تمرین های زیر نصب کن و پکیج ها مربوط به تمرین رو داخل این virtualenv نصب کن.

# تمرین

  1. یک اسکریپت بنویس که هر ۱۰ ثانیه یک بار سایتی رو چک می‌کنه و اگر نسبت به دفعه قبلیش تغییر کرده باشه، یک پیغام چاپ کنه که سایت تغییر کرد. برای تست سایتی که عوض نمی‌شه می‌تونی از سایت ویکی‌پدیا استفاده کنی. (از پکیج requests استفاده کن)
  2. یک برنامه بنویس که بهش یک زمان می‌دیم و بهمون می‌گه کدوم کشور زمانش به زمانی که دادیم نزدیک‌تره.
  3. برنامه‌ای بنویس که یک عکس راه‌راه با ۲ رنگ قرمز و آبی تولید کنه.

# آشنایی با Regex

Regular Expression یا عبارت منظم یا Regex یه زبان برای توصیف یه الگو برای جستجو توی متنه مثلا با regex میتونیم بگردیم هرچی email توی یه متن طولانی پیدا میکنیم رو پیدا کنیم. توی web development چه سمت backend و چه سمت frontend خیلی از regex استفاده می‌شه مثلا برای اینکه چک کنیم ورودی کاربر درسته یا نه.

  • کتابخونه‌ی re از python رو توی اینترنت پیدا کن و مستندا
